MIPS Within MARS SIMULATOR
1. First unzip the folder to a specified location
2. Then open MIPS MARS SIMULATOR
3. Click Open file within MIPS
4. Select the desired MIPS .asm file to run
5. Once opened, press build to compile program
6. Finally press Run button to run the program.



X86 Intel on Windows 32-Bit
1. First unzip the folder to a specified location
2. Then open up Microsoft Visual Studio
3. Afterwards select project solution to open. 
4. Select ADIL_TAKE_HOME_TEST_2.sln
5. Once opened click build then build solution.
6. Set a breakpoint
7. Finally press Debug-> start Debugging
8. You can Press Debug -> Windows -> registers
9. You can Press Debug -> Windows -> memory -> memory 1



Linux 64-Bit with GCC and GDB
1. First unzip the folder to a specified location
2. Then open up terminal
3. change directory into specified directory
4. Then using GCC, use the following command, gcc -g -o0 FILENAME.c -o FILENAME
5. Then to use GDB use GDB FILENAME
6. break main
7. run
8. disassembly
